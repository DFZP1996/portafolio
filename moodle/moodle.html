01

El Problema y la Solución

El Reto

La Secretaría de Salud presentaba una alta deserción en la oferta formativa virtual gratuita. Factores sociodemográficos y académicos no estaban siendo monitoreados de manera unificada, lo que impedía prevenir el abandono a tiempo o entender el perfil real de los estudiantes.

El MVP

Desarrollamos un Producto Mínimo Viable con dos frentes: 1. Dashboard Analítico interactivo para que los gestores consoliden KPIs y tomen decisiones. 2. Módulo de IA Adaptativa para generar recomendaciones automáticas al estudiante según su perfil inicial.

02

Arquitectura de Datos & Wireframes

Antes de aplicar color y tipografía, el proceso creativo inició con la estructuración lógica de los datos. Se diseñó un esquema (wireframe) modular, dividiendo la carga cognitiva en bloques de información digeribles para los administradores.

Modelado de Información

1. KPIs Globales (Cursos, Matrículas)
2. Variables Demográficas (Sexo, Edad)
3. Métricas Operativas (Aprobación)
4. Datos Geográficos (Comunas)
5. Resultados Encuesta de Satisfacción
03

Personalización UI/UX

Para garantizar la accesibilidad y adaptarse a las preferencias cognitivas de los gestores de salud, el dashboard incorpora un sistema de Multitema en tiempo real construido con variables CSS nativas. Esto permite a los usuarios modificar los contrastes y las paletas de toda la plataforma con un solo clic, reduciendo la fatiga visual.

Analítica en la palma de la mano

La interfaz fue construida bajo un enfoque Mobile-First, permitiendo a los gestores revisar la analítica desde cualquier dispositivo sin perder claridad visual ni interactividad.

A través de un prototipo conversacional, la plataforma recopila información diagnóstica inicial del usuario. Estos datos alimentan la Inteligencia Artificial Adaptativa para generar recomendaciones y seguimiento en tiempo real directamente en la vista del celular.

El proyecto completo está vivo

Las interfaces mostradas aquí son solo un entorno simulado para el portafolio. El proyecto final, completamente codificado y conectado a bases de datos anonimizadas, está listo para ser revisado a profundidad durante una entrevista técnica.
